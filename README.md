# 🚀 Resume Screening using Machine Learning (Task 3)

This project is part of my **Machine Learning Internship at Future Interns**.

## 📌 Objective
To build a machine learning model that can automatically classify resumes into different job categories using NLP techniques.

---

## 🔍 Features
- Text preprocessing and cleaning
- Feature extraction using TF-IDF
- Resume classification using ML algorithms
- Candidate ranking based on similarity score

---

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- NLP (TF-IDF)

---

## 📊 Visualizations
- Resume count by job category
- Top skills in dataset
- Candidate ranking (Composite Fit Score)
- Similarity score analysis

---

## 📈 Results
The model was able to classify resumes and rank candidates based on similarity with job descriptions.
